# <a name="endif-statements-are-no-longer-supported-use-end-if-instead"></a>Instrukcje "EndIf" nie są już obsługiwane; Użyj polecenia "End If"

`EndIf`Instrukcja nie jest już obsługiwana.  
  
 **Identyfikator błędu:** BC30826  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj `End If` , aby zakończyć `If` instrukcje.
