# <a name="statement-cannot-appear-outside-of-a-method-bodymultiline-lambda"></a>Instrukcja nie może występować poza treścią metody/wielowierszowym wyrażeniem lambda

Instrukcje wykonywalne muszą być używane w ramach procedur.  
  
 **Identyfikator błędu:** BC30689  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przenieś instrukcję do procedury lub podprocedury.
