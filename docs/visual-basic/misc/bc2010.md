# <a name="compilation-failed-message"></a>Kompilacja nie powiodła się: \<message>

Kompilacja nie powiodła się z powodu okoliczności określonych w komunikacie o błędzie  
  
 **Identyfikator błędu:** BC2010  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Sprawdź komunikat o błędzie, aby określić dokładną przyczynę błędu.
