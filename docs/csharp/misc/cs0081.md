# <a name="compiler-error-cs0081"></a><span data-ttu-id="db729-103">Błąd kompilatora CS0081</span><span class="sxs-lookup"><span data-stu-id="db729-103">Compiler Error CS0081</span></span>
<span data-ttu-id="db729-104">Deklaracja parametru typu musi być identyfikatorem, a nie typem</span><span class="sxs-lookup"><span data-stu-id="db729-104">Type parameter declaration must be an identifier not a type</span></span>  
  
 <span data-ttu-id="db729-105">Podczas deklarowania metody ogólnej lub typu należy określić parametr typu jako identyfikator, na przykład "T" lub "InputType".</span><span class="sxs-lookup"><span data-stu-id="db729-105">When you declare a generic method or type, specify the type parameter as an identifier, for example "T" or "inputType".</span></span> <span data-ttu-id="db729-106">Gdy kod klienta wywołuje metodę, dostarcza typ, który zastępuje każde wystąpienie identyfikatora w metodzie lub treści klasy.</span><span class="sxs-lookup"><span data-stu-id="db729-106">When client code calls the method, it supplies the type, which replaces each occurrence of the identifier in the method or class body.</span></span> <span data-ttu-id="db729-107">Aby uzyskać więcej informacji, zobacz [parametry typu ogólnego](../programming-guide/generics/generic-type-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="db729-107">For more information, see [Generic Type Parameters](../programming-guide/generics/generic-type-parameters.md).</span></span>  
  
```csharp  
// CS0081.cs  
class MyClass  
{  
   public void F<int>() {}   // CS0081  
   public void F<T>(T input) {}   // OK  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
      a.F<int>(2);  
      a.F<double>(.05);  
   }  
}  
```
