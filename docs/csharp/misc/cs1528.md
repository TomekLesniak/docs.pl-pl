<span data-ttu-id="c11a7-104">Oczekuj or = (nie można określić argumentów konstruktora w deklaracji)</span><span class="sxs-lookup"><span data-stu-id="c11a7-104">Expected ; or = (cannot specify constructor arguments in declaration)</span></span>  
  
 <span data-ttu-id="c11a7-105">Odwołanie do klasy zostało sformułowane tak, jakby obiekt klasy był tworzony.</span><span class="sxs-lookup"><span data-stu-id="c11a7-105">A reference to a class was formed as if an object to the class was being created.</span></span> <span data-ttu-id="c11a7-106">Na przykład podjęto próbę przekazania zmiennej do konstruktora.</span><span class="sxs-lookup"><span data-stu-id="c11a7-106">For example, there was an attempt to pass a variable to a constructor.</span></span> <span data-ttu-id="c11a7-107">Użyj operatora [New](../language-reference/operators/new-operator.md) , aby utworzyć obiekt klasy.</span><span class="sxs-lookup"><span data-stu-id="c11a7-107">Use the [new](../language-reference/operators/new-operator.md) operator to create an object of a class.</span></span>  
  
 <span data-ttu-id="c11a7-108">Poniższy przykład generuje CS1528:</span><span class="sxs-lookup"><span data-stu-id="c11a7-108">The following sample generates CS1528:</span></span>  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```
