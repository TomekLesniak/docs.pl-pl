# <a name="compiler-error-cs0611"></a>Błąd kompilatora CS0611

Elementy tablicy nie mogą być typu "Type"  
  
 Istnieją typy, których nie można używać jako typu tablicy. Te typy obejmują **System. TypedReference** i **System. ArgIterator**.  
  
 Poniższy przykład generuje CS0611 w wyniku użycia elementu **System. TypedReference** jako element tablicy:  
  
```csharp  
// CS0611.cs  
public class a  
{  
   public static void Main()  
   {  
      System.TypedReference[] ao = new System.TypedReference [10];   // CS0611  
      // try the following line instead  
      // int[] ao = new int[10];  
   }  
}  
```
