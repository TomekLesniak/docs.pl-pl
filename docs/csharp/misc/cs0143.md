# <a name="compiler-error-cs0143"></a>Błąd kompilatora CS0143

Typ "Class" nie ma zdefiniowanych konstruktorów  
  
 Nie jest dostępny żaden odpowiedni Konstruktor. Jest to przypadek dla wbudowanych typów wartości liczbowych, które są inicjowane po prostu przez przypisanie do nich wartości.  
  
 Poniższy przykład generuje CS0143:  
  
```csharp  
// CS0143.cs  
class MyClass  
{  
   static public void Main ()  
   {  
      double d = new double(4.5);   // CS0143  
      // Try this line instead:  
      // double d = 4.5;  
   }  
}  
```
