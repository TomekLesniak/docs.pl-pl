# <a name="compiler-warning-level-1-cs0684"></a>Ostrzeżenie kompilatora (poziom 1) CS0684
Interfejs "Interface" oznaczony atrybutem "CoClassAttribute" nie jest oznaczony atrybutem "ComImportAttribute"  
  
 W przypadku określenia **klasy** CoClassAttribute w interfejsie należy również określić **ComImportAttribute**.  
  
 Poniższy przykład generuje CS0684:  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
