# <a name="compiler-error-cs1650"></a><span data-ttu-id="38801-103">Błąd kompilatora CS1650</span><span class="sxs-lookup"><span data-stu-id="38801-103">Compiler Error CS1650</span></span>
<span data-ttu-id="38801-104">Nie można przypisać pól statycznego pola tylko do odczytu "identifier" (z wyjątkiem: w konstruktorze statycznym lub inicjatorze zmiennych).</span><span class="sxs-lookup"><span data-stu-id="38801-104">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="38801-105">Ten błąd występuje podczas próby zmodyfikowania elementu członkowskiego pola, które jest tylko do odczytu i statycznego, w którym nie można go modyfikować.</span><span class="sxs-lookup"><span data-stu-id="38801-105">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="38801-106">Aby rozwiązać ten problem, Ogranicz przydziały do pól tylko do odczytu do konstruktora lub inicjatora zmiennych lub Usuń `readonly` słowo kluczowe z deklaracji pola.</span><span class="sxs-lookup"><span data-stu-id="38801-106">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```csharp  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```
