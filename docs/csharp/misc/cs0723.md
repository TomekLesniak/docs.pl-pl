# <a name="compiler-error-cs0723"></a>Błąd kompilatora CS0723
Nie można zadeklarować zmiennej typu statycznego "Type"  
  
 Nie można tworzyć wystąpień typów statycznych.  
  
 Poniższy przykład generuje CS0723:  
  
```csharp  
// CS0723.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC sc = null;  // CS0723  
   }  
}  
```
