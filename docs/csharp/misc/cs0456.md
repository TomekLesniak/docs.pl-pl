# <a name="compiler-error-cs0456"></a><span data-ttu-id="4b51b-103">Błąd kompilatora CS0456</span><span class="sxs-lookup"><span data-stu-id="4b51b-103">Compiler Error CS0456</span></span>
<span data-ttu-id="4b51b-104">Parametr typu "Type" o nazwie "Structure" ma ograniczenie "struct", dlatego nie można użyć parametru typu "name" jako ograniczenia dla parametru "type name 2".</span><span class="sxs-lookup"><span data-stu-id="4b51b-104">Type parameter 'Type Parameter Name 1' has the 'struct' constraint so 'Type Parameter Name 1' cannot be used as a constraint for 'Type Parameter Name 2'</span></span>  
  
 <span data-ttu-id="4b51b-105">Ograniczenia dotyczące typu wartości są zapieczętowane niejawnie, dlatego nie można używać tych ograniczeń jako ograniczeń dla drugiego parametru typu.</span><span class="sxs-lookup"><span data-stu-id="4b51b-105">Value type constraints are implicitly sealed so those constraints cannot be used as constraints on a second type parameter.</span></span> <span data-ttu-id="4b51b-106">Dzieje się tak, ponieważ typy wartości nie mogą zostać zastąpione.</span><span class="sxs-lookup"><span data-stu-id="4b51b-106">This is because value types cannot be overridden.</span></span> <span data-ttu-id="4b51b-107">Aby rozwiązać ten problem, należy umieścić ograniczenie typu wartości bezpośrednio w drugim parametrze typu, zamiast wykonać to pośrednio przy użyciu pierwszego parametru typu.</span><span class="sxs-lookup"><span data-stu-id="4b51b-107">To resolve this error, put a value type constraint directly on the second type parameter, instead of doing so indirectly by means of the first type parameter.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4b51b-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="4b51b-108">Example</span></span>  
 <span data-ttu-id="4b51b-109">Poniższy przykład generuje CS0456.</span><span class="sxs-lookup"><span data-stu-id="4b51b-109">The following sample generates CS0456.</span></span>  
  
```csharp  
// CS0456.cs  
// compile with: /target:library  
public class GenericsErrors  
{  
   public class G5<T> where T : struct  
   {

      public class N2<U> where U : struct {}   // OK  
   }  
}  
```
