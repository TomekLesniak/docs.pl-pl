# <a name="c-tutorials"></a>Samouczki języka C#

Witamy w samouczkach dotyczących języka C#. Rozpoczynają się one od interaktywnych lekcji, które można uruchomić w przeglądarce. Nowsze samouczki i bardziej zaawansowane samouczki ułatwiają współpracę z narzędziami deweloperskimi platformy .NET w celu tworzenia programów w języku C# na komputerze.

## <a name="introduction-to-c-interactive-tutorials"></a>Wprowadzenie do interaktywnych samouczków języka C#

Jeśli chcesz rozpocząć Eksplorowanie w formacie wideo, [Seria filmów w języku c# 101](https://aka.ms/dotnet3-csharp) zawiera wprowadzenie do języka c#. W tych samouczkach znajdziesz informacje o pojęciach, które można poznać.

Pierwsze lekcje objaśniają koncepcje języka C# przy użyciu małych fragmentów kodu. Poznasz podstawy składni języka C# i sposób pracy z typami danych, takimi jak ciągi, liczby i wartości logiczne. Jest to wszystko interaktywne, a ty będziesz pisać i uruchamiać kod w ciągu kilku minut. W tych pierwszej lekcji nie założono wcześniej znajomości programowania ani języka C#.
